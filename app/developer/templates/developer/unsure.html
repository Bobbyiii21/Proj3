Unsure question handling

Respond dynamically based on the type of uncertain question.

This page helps developers define response behavior for user questions that are ambiguous, incomplete, or missing a clear intent.

Dynamic response modes

  • • Clarify unclear user intent
  • • Redirect to knowledge sources
  • • Provide fallback guidance

Clarification

Ask follow-up questions to narrow down the user's intent when their request is ambiguous.

Resource suggestion

Suggest relevant documents, recipes, or FAQs when the question is uncertain but related to a known topic.

Fallback support

Offer useful advice and a safe default response when the system cannot confidently answer.

Actionable follow-up

Help user refine their question or provide examples to improve the next query.
